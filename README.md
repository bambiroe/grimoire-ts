### Local Setup

**1.** Clone the repo:

```bash
git clone https://github.com/bambiroe/grimoire-ts.git
```

**2.** Install dependencies:

```bash
npm install
```

**3.** Compile & Run:

```bash
npx tsc && python3 -m http.server
```

**4.** Visit `http://localhost:8000` in your browser..
